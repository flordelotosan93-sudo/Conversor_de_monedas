Conversor de Monedas - Java Console App

Este proyecto es un desafío propuesto por el programa ONE. Se trata de una aplicación de consola desarrollada en Java, diseñada para facilitar la conversión de divisas de manera interactiva y precisa.

La aplicación utiliza la ExchangeRate-API, lo que permite obtener tasas de cambio en tiempo real, garantizando que las conversiones reflejen los valores actuales del mercado. Los usuarios pueden seleccionar entre varias opciones de conversión predefinidas, incluyendo dólares estadounidenses, pesos argentinos, pesos mexicanos y reales brasileños.
